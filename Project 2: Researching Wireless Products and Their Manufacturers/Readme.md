Wireless Products Research Assignment (2026)

1. LoRaWAN Gateway
   Manufacturer: Kerlink

Link:

Description: This outdoor LoRaWAN gateway is designed to provide long-range, low-power connectivity for IoT sensors in challenging environments. It features a carrier-grade design with high interference resistance and supports integrated cellular backhaul for seamless data transmission.

2. Zigbee Module
   Manufacturer: Silicon Labs

Link:

Description: This module is a high-performance wireless solution designed for mesh networking in smart homes and industrial automation. It offers industry-leading security (PSA Level 3) and ultra-low power consumption to extend the battery life of end-node devices.

3. Wi-Fi 6 Chipset
   Manufacturer: Qualcomm

Link:

Description: This chipset provides the core processing power for high-density Wi-Fi 6 networks, supporting advanced features like MU-MIMO and OFDMA. It is engineered to deliver multi-gigabit speeds and consistent performance in environments with hundreds of concurrent connected devices.

4. NB-IoT Module
   Manufacturer: Quectel Wireless Solutions

Link:

Description: The BC660K-GL is an ultra-compact, high-performance NB-IoT module that supports a wide range of frequency bands for global deployment. Its extremely low power consumption makes it ideal for massive IoT applications such as smart metering, asset tracking, and environmental monitoring.

5. Sigfox Device
   Manufacturer: STMicroelectronics

Link:

Description: This device is a sub-1GHz RF transceiver specifically optimized for Sigfox networks to enable ultra-long-range communication with minimal energy usage. It is highly integrated, reducing the overall bill of materials (BOM) for manufacturers building small, battery-operated IoT sensors.

6. Bluetooth Low Energy (BLE) Beacon
   Manufacturer: Nordic Semiconductor

Link:

Description: This SoC serves as the brain for advanced BLE beacons, enabling precise indoor positioning and proximity-based data broadcasting. It features a powerful ARM Cortex-M4 processor and very low peak current, allowing beacons to run for years on a single coin-cell battery.

7. Ultra-Wideband (UWB) Transceiver
   Manufacturer: Qorvo

Link:

Description: This UWB transceiver is designed for high-precision real-time location systems (RTLS), offering centimeter-level accuracy for distance measurement. It supports enhanced security features for secure "bubble" applications, such as hands-free keyless entry and secure mobile payments.

8. Wi-Fi 6E Access Point
   Manufacturer: Aruba (a Hewlett Packard Enterprise company)

Link:

Description: This access point utilizes the 6 GHz band to provide wider channels and reduced interference compared to traditional Wi-Fi. It is designed for enterprise environments that require massive throughput and low latency for AR/VR and high-definition video applications.

9. 5G IoT Module
   Manufacturer: Telit Cinterion

Link:

Description: This module enables high-speed 5G connectivity for industrial routers and IoT gateways, supporting both Sub-6 GHz and mmWave spectrums. It features integrated GNSS for location tracking and is designed to handle the high data demands of 4K surveillance and autonomous robotics.
